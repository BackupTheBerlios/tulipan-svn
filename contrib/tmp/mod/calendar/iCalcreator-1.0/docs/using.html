Description:

iCalcreator is a PHP implementation of RFC2445/RFC2446 to create iCal/xCal formatted files.

1. INTRO

iCalcreator is a PHP class creating iCal formatted files, iCal/xCal, for non-calendar systems like CMS, project management systems and other applications able to produce calendar information like agendas, tasks, reports, totos, journaling data and for input to calendar systems and applications.

iCalcreator is built of a single class file with a simple interface and are calendar component property oriented.

No controls between calendar components/properties are made.

Knowledge of calendar protocol rfc2445 - Internet Calendaring and Scheduling Core Object Specification (iCalendar) AND rfc2446 - iCalendar Transport-Independent Interoperability Protocol (iTIP) Scheduling Events, BusyTime, To-dos and Journal Entries is necessary! All functions calls are made as simple as possible BUT (, !!!,) read these rfc's properly!

RFC2445 - Internet Calendaring and Scheduling Core Object Specification (iCalendar) in text and HTML format.
RFC2446 - iCalendar Transport-Independent Interoperability Protocol (iTIP) Scheduling Events, BusyTime, To-dos and Journal Entries in text format.

xCal (iCal xml) format is supported but still experimental.

Examples how to employ iCalcreator in software development:
Create iCal event file on-the-fly
Create complex iCal file on-the-fly

There are free iCal/xCal icons in the images directory, to use as buttons on a webpage.
